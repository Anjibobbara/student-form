# 🎓 Student Registration System

A full-stack Student Registration Management System built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## 🚀 Features

- Student Registration Form
- Store Student Details in MongoDB
- View Registered Students
- Admin Dashboard
- Responsive Design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose

## 📂 Project Structure

```
student-form/
│── models/
│── public/
│── server.js
│── package.json
│── README.md
```

## ⚙️ Installation

Install dependencies:

```bash
npm install
```

Start the server:

```bash
node server.js
```

Open your browser:

```
http://localhost:5000
```
